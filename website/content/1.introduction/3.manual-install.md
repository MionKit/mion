## Server Side

Install the router and Configure `tsConfig.json` to enable [runTypes metadata.](../1.introduction/1.about-mion.md#automatic-serialization-validation)

```bash
npm i @mionjs/core @mionjs/router
npm i -D @mionjs/devtools
```

```json [tsConfig.json]
{
  "compilerOptions": {
    // typescript compiler options ...
  },
  // enables runtime type metadata for mion's validation and serialization
  "reflection": true
}
```

Install dependencies depending on where you running mion.<br>For local development use Node or Bun.

::code-group
```bash [Node.js]
npm install @mionjs/platform-node
```

```bash [Bun]
npm install @mionjs/platform-bun
```

```bash [AWS Lambda]
npm install @mionjs/platform-aws
```

```bash [Google Cloud Functions]
npm install @mionjs/platform-gcloud
```
::
